# Earnings Call Intelligence Platform

A production-grade, real-time, event-driven platform that automatically **discovers, records,
transcribes, and publishes earnings-call webcasts** for publicly traded companies — delivering
near-live, searchable transcripts at scale.

> Bootstrapping. See the master design at `docs/superpowers/specs/` and the feature build order
> under `features/`. Full README (architecture, quickstart, make targets) lands in feature `B3`.
